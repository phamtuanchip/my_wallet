# NFC Wallet App Icon

## Icon Description
The app icon features a modern smartphone with NFC card symbols and antenna waves, representing the dual functionality of reading and emulating NFC cards.

### Design Elements:
- **Smartphone silhouette** in white on blue background
- **NFC antenna waves** emanating from the device
- **Virtual NFC card** displayed on the phone screen
- **Card chip pattern** showing the contactless payment symbol
- **"NFC WALLET" text** at the bottom
- **Color scheme**: Blue (#1976D2) background with white elements

### Icon Specifications:
- **Format**: PNG (512x512 recommended)
- **Background**: Solid blue (#1976D2)
- **Style**: Modern, clean, professional
- **Platforms**: Android, iOS, Web, Windows

## How to Create the Icon:

1. **Use an online icon generator** like:
   - [App Icon Generator](https://appicon.co/)
   - [Icon Kitchen](https://icon.kitchen/)
   - [Figma](https://figma.com) or [Canva](https://canva.com)

2. **Design the icon** with the following elements:
   - Smartphone outline
   - NFC waves (3 curved lines)
   - Card symbol on screen
   - "NFC WALLET" text

3. **Generate platform-specific icons**:
   ```bash
   flutter pub run flutter_launcher_icons
   ```

4. **Test on device**:
   ```bash
   flutter run --debug -d <device-id>
   ```
